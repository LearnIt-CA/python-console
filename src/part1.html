Brute-Force Algorithm Overview

Logic Plan

  • Initialize a counter for the total number of guesses.
  • Define the PIN length and compute the keyspace (all possible combinations).
  • Iterate through every candidate PIN one by one.
    • Increment the attempt counter.
    • Compare the current guess with the secret PIN.
    • Stop immediately when the correct PIN is found.
  • Report the discovered PIN and total number of attempts.
