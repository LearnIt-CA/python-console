02_VARIABLES

Variables store data that can be used and modified throughout your code. Think of them as containers with labels.

VARIABLE SYNTAX 
#Example:
name = "Alex"              
age = 16                        
print(name, "is", age)

03_BASIC_OPERATIONS

Python supports various operations for manipulating data:

OPERATIONS 
#Example 1:
x = 9 
y = "nine"
print(9 + 2)
print(9 / 2)
print(x * y)

#Example 2:

04_FUNCTIONS

Functions are reusable blocks of code that perform specific tasks:

FUNCTION DEFINITION & USAGE 
#Example:
def greet(name):
    print("Hello,", name)
greet("Alice")

05_CONDITIONALS

Conditional statements allow your program to make decisions:

IF-ELSE STATEMENTS 
#Example:
age = 16
if age >= 18:
    print("You can vote!")
else:
    print("You're too young to vote.")

06_LOOPS

Loops execute a block of code multiple times:

LOOP STRUCTURES 
#Example:

    print("Hello!", i)
